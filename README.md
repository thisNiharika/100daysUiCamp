🚀 100 – NextGen UI Challenge  
100 Days to FrontEnd Glory  

An interactive, no login progress tracker designed to help developers complete a 100 day frontend challenge with style.

Built with a vivid gradient UI, smooth interactions, and smart progress tracking — this project focuses on clean UX, motivation, and daily consistency.



✨ Features

🎨 Vivid Gradient UI  
📊 Live Progress Tracking (0–100%)  
📅 Jump to Day Navigation  
✅ Mark Tasks as Completed  
🗂️ Week wise Organization (Week 1 – Days 1–10, etc.)  
🖨️ Print / Save as PDF Option  
🔓 No Login Required (Client side only)  



🧠 How It Works

Each day contains a frontend challenge or task.

Users can:

Select any day using the bottom day selector.  
Mark a task as complete.  
See progress update automatically.  
Print or save progress as a PDF.  

All tracking is handled on the client side using LocalStorage.



🛠️ Tech Stack

HTML5  
CSS3 (Modern UI / Gradients)  
JavaScript (Vanilla JS)  
LocalStorage for progress persistence
